Les bases en HTML

Dans un document html, on retrouve comme structure de base:

  • doctype : correspondant au type de document
  • html : correspondant à la langue
  • head : correspondant à tous les metadonnées presente dans le document
  • body : coorespondant au corps du document

Pour l'entête notre structure de base est constitué de cette façon :

  1. La balise Meta dans laquelle on a rentré des données comme le type de caractere ou des mots clé
  2. Le titre du document
  3. Le style de la page comme le fond etc
